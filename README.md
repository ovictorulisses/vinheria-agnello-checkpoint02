# Vinheria Agnello - Sistema de Avaliação de Estoque e Tendência

### 🍷 Descrição do Projeto
Sistema de cadastro, avaliação de estoque e classificação de vinhos artesanais da Vinheria Agnello. Desenvolvido como parte do Checkpoint 02 de JavaScript da FIAP, aprofundando o uso de funções reutilizáveis, escopos de variáveis e laços de repetição.

## 🧠 Objetivo do Projeto
O objetivo deste sistema é aplicar conceitos fundamentais de JavaScript para gerenciar e avaliar dados de vinhos artesanais, focando em:
*   Funções reutilizáveis e organização de código.
*   Escopo correto de variáveis (local e global).
*   Laços de repetição e iteração (`for`, `while`).
*   Validação de entradas do usuário.
*   Interação com o usuário via `prompt()` e `alert()`.
*   Exibição de dados organizados no `console.log()`.

## ⚙️ Funcionalidades
*   Cadastro de múltiplos vinhos em sequência.
*   Validação de dados de entrada (nome, safra, quantidade).
*   Verificação automática de estoque baixo (menos de 5 unidades).
*   Classificação do vinho por idade da safra (jovem, amadurecido, antigo).
*   Resumo final com total de cadastros, vinhos com estoque baixo e safra mais antiga.
